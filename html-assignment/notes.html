Lists Run Down Town

A mental map of how to use the following tags for myself to remember the contextual uses here.

<ol> Ordered List! Exactly as it sounds since it uses numbers.

<ul> Unordered List! The exact opposite of an ordered list with no numbers or priority.

<li> This is the bread and butter that actually puts in either the numbers or dash/dots for the list. You can use several and they are generally placed within either an ordered list or unordered list.

<dl> Description List! Pretty much sets the boarders for a description tag.

<dt> Description Terms! Pretty much generally used for the title!. .

<dd> Description Details! Where you'd pull the details and real grit of it all, duh. .

HTML Mental Gymnastics Sum Up

A mental map of how to use the following tags for myself to remember the contextual uses here. P.S. Note to self, don't forget about <Header> when choosing what to use!

<Main> can only be used once on a page and cannot be a child of most elements like the following (article, footer, header, nav), and is good to use if you're working with accessability.

<Div> Even more general then section, it's great for miscellaneous use.

<Article> For independent content. E.g. Blog posts, news articles, forums.

<Section> More general, can be used for anything that needs to be sectioned off. .
